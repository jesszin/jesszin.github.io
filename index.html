Wise Data

Wise Data(data visualization)

참여 과정
Python기반 tool Pycharm으로 데이터 시각화에 필요한 그래프를 그리기 위한 node(꼭지점) , edge(연결선)를 생성한 뒤,
gephi로 연결 관계를 정의 할 수 있도록 그래프를 생성하여 관련성을 돋보일 수 있게 배치한 뒤, svg로 저장합니다.
저장한 svg 파일을 illustrator로 학과끼리 묶어주는 시각화 작업을 합니다.
스무개 이상의 대학교 학과의 데이터 시각화 작업을 마쳤습니다.


각 대학교의 학과 간의 관련성을 보여주는 학과 지도입니다.
학과의 데이터 시각화에 필요한 node,edge 생성 및 시각화 작업 전부 참여하였습니다.
Type : Data visualization
Tool : PyCharm , Gephi , Illustrator
